# P1_AED2_26-1
Projeto da disciplina Algoritmos e Estruturas de Dados II. 
Discentes: Felipe Gabriel e Laíze Suélia
Docente: Ivanovitch Silva

O presente projeto tem como objetivo analisar documentos referentes a processos movidos pelo Ministério Público de Contas do Estado de São Paulo, onde são apontados graves riscos em investimentos milionários envolvendo o Instituto de Previdência Municipal e o Banco Master. Para essa análise serão utilizados os conceitos aprendidos em sala de aula sobre grafos utilizando a biblioteca NetworkX, bem como utilizar conceitos de Processamento de Linguagem Natural utilizando o recurso do Named Entity Recognition (NER).
