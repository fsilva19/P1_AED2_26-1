## Discentes: Felipe Gabriel e Laíze Suélia </br> Docente: Ivanovitch Silva

## Grafos e Named Entity Recognition (NER)
<p> O presente projeto tem como objetivo analisar documentos referentes a processos movidos pelo Ministério Público de Contas do Estado de São Paulo, onde são apontados graves riscos em investimentos milionários envolvendo o Instituto de Previdência Municipal e o Banco Master. Para essa análise serão utilizados os conceitos aprendidos em sala de aula sobre grafos utilizando a biblioteca NetworkX, bem como utilizar conceitos de Processamento de Linguagem Natural utilizando o recurso do Named Entity Recognition (NER). </p>

> As informações referentes aos processos foram extraídas do site oficial do [Ministério Público de Contas do Estado de São Paulo](https://www.mpc.sp.gov.br/procuradoria-do-mpc-sp-aponta-graves-riscos-em-investimentos-milionarios-feitos-por-instituto-de).
